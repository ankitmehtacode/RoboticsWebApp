## Installation Instructions

1. Create a new folder: `sun-robotics-ai`
2. Copy all files below into their respective locations
3. Run: `npm install`
4. Run: `npm run dev`

## Dependencies Required
- React 18.3.1
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
